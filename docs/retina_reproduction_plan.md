# 视网膜模块代码复现计划

## 概述

本计划旨在复现 `RetinaScripts/` 模块的完整分析流程，该模块对视网膜神经节细胞（RGC）在白噪声棋盘格刺激下的响应进行系统性分析，逐步构建从简单到复杂的神经编码模型。

**论文参考：** Aljadeff, Lansdell, Fairhall & Kleinfeld (2016) — *Analysis of Neuronal Spike Trains, Deconstructed*

---

## 阶段一：数据准备与预处理

### 步骤 1.1 — 构建刺激矩阵与响应向量
**对应脚本：** `script_101_RetinaData_BuildStimulus.m`

**目标：** 从原始二进制文件中提取并组织刺激和响应数据

**输入数据：**
- `whitenoise.raw` — 原始棋盘格刺激（二进制格式）
- `whitenoisec1.isk ~ whitenoisec53.isk` — 53 个细胞的脉冲序列
- `RetinaCellParameters_*.mat` — 各细胞参数（空间感受野位置、大小、时间延迟等）

**关键操作：**
1. 使用 `ReadFramev2.m` 读取原始刺激文件，提取每个细胞对应的空间感受野区域
2. 将刺激重塑为 (T × N_X) 矩阵，T 为时间帧数，N_X 为空间维度
3. 对每个像素维度进行零均值化和归一化
4. 根据细胞的时间延迟参数用 `circshift` 对齐刺激和响应
5. 对于多帧时间依赖（NT > 1），构建时空联合刺激矩阵
6. 将数据拆分为训练集和测试集
7. 进行 Jackknife 分组用于后续交叉验证

**输出：** 预处理后的刺激矩阵 `S_train/S_test` 和响应向量 `R_train/R_test` 的 `.mat` 文件

**复现要点：**
- 理解 `ReadFramev2.m` 的二进制文件读取格式
- 注意刺激的空间裁剪和时间延迟对齐
- 默认分析对象为 cell 3

## 阶段二：线性模型 — STA

### 步骤 2.1 — 计算脉冲触发平均 (STA)
**对应脚本：** `script_102_RetinaData_STA.m`

**目标：** 估计神经元的线性感受野

**方法：**
$$\text{STA} = \frac{1}{n_{\text{spikes}}} \sum_{t: \text{spike at } t} \mathbf{s}(t)$$

**关键操作：**
1. 对每个 Jackknife 折的训练数据计算 STA
2. STA 即为触发脉冲时刻对应的刺激向量的平均值
3. 对白噪声刺激，STA 是最优线性滤波器的无偏估计

**输出：** 每个 Jackknife 折的 STA 向量

**复现要点：**
- STA 计算直接、无需优化
- 注意刺激已预先白化（零均值、单位方差），因此 STA ∝ 刺激-响应互相关

## 阶段三：二阶模型 — STC

### 步骤 3.1 — STC 显著性检验
**对应脚本：** `script_103_RetinaData_STC_significance.m`

**目标：** 计算脉冲触发协方差矩阵并确定显著特征方向

**方法：**
$$C_{\text{STC}} = \frac{1}{n_{\text{spikes}}} \sum_{t: \text{spike at } t} (\mathbf{s}(t) - \text{STA})(\mathbf{s}(t) - \text{STA})^T - C_{\text{prior}}$$

**关键操作：**
1. 计算脉冲触发协方差矩阵（STC）
2. 对 STC 进行特征值分解
3. 将特征值与 Marchenko-Pastur 随机矩阵分布进行比较
4. 确定统计显著的特征方向（超出随机矩阵理论预测范围的特征值）

**输出：** STC 特征值、特征向量、显著性判断

### 步骤 3.2 — 构建 STC 模型
**对应脚本：** `script_104_RetinaData_STC_model.m`

**目标：** 使用显著 STC 特征方向构建 LN 模型

**关键操作：**
1. 选择通过显著性检验的特征方向
2. 将刺激投影到这些方向上
3. 使用 Bayes 定理估计非线性函数：$P(\text{spike}|\text{projection})$
4. 验证模型在测试集上的预测性能

**输出：** STC 模型参数（显著特征方向 + 非线性函数）

**复现要点：**
- Marchenko-Pastur 分布作为零假设
- 正/负显著特征值分别对应激励/抑制子空间

## 阶段四：最大噪声熵模型 — MNE

### 步骤 4.1 — MNE 模型拟合
**对应脚本：** `script_105_RetinaData_MNE_fitting.m`

**目标：** 拟合参数化的二阶 LN 模型

**方法（论文公式 32）：**
$$P(\text{spike}|\mathbf{s}) = \frac{1}{1 + \exp\left(-(A + \mathbf{H}^T\mathbf{s} + \mathbf{s}^T\mathbf{J}\mathbf{s})\right)}$$

其中：
- $A$ — 标量偏置
- $\mathbf{H}$ — 线性特征向量
- $\mathbf{J}$ — 二次特征矩阵（对称）

**关键操作：**
1. 计算约束量：脉冲概率 $p_{sp}$、一阶刺激-响应相关、二阶相关
2. 初始化参数：$A$ 从 log-odds 获得，$\mathbf{H}$ 和 $\mathbf{J}$ 随机初始化
3. 使用共轭梯度法（`frprmn_global_min`）最小化对数损失
4. 在测试集上进行 early stopping 防止过拟合

**辅助函数：**
- `MNEfit_RetinaData.m` — 主拟合函数
- `logloss.m` — 计算对数损失目标函数
- `dlogloss.m` — 计算梯度
- `frprmn_global_min.m` — 共轭梯度优化器（含 early stopping）

**输出：** 每个 Jackknife 折的 MNE 模型参数

### 步骤 4.2 — MNE 模型后处理
**对应脚本：** `script_106_RetinaData_MNE_model.m`

**目标：** 提取显著的二次特征方向

**关键操作：**
1. 对各 Jackknife 折的拟合结果取平均
2. 提取模型参数 $A$, $\mathbf{H}$, $\mathbf{J}$
3. 对 $\mathbf{J}$ 进行特征值分解（公式 34-35）
4. 通过置换检验（500 次重排 $\mathbf{J}$ 的对角/非对角元素）构建零分布
5. 确定统计显著的特征值/特征方向

**输出：** 显著 MNE 特征方向、特征值、零分布

**复现要点：**
- MNE 是 STC 的参数化替代，可同时估计线性和二次特征
- 共轭梯度法的 early stopping 对防止过拟合至关重要
- 置换检验用于确定显著性阈值

## 阶段五：广义线性模型 — GLM

### 步骤 5.1 — GLM 拟合
**对应脚本：** `script_107_RetinaData_GLM.m`

**目标：** 拟合包含脉冲历史的 GLM

**模型结构：**
$$\lambda(t) = \exp\left(\mathbf{k}^T \mathbf{s}(t) + \mathbf{h}^T \mathbf{r}_{\text{hist}}(t) + b\right)$$

其中：
- $\mathbf{k}$ — 刺激滤波器（时空双线性参数化）
- $\mathbf{h}$ — 脉冲后历史滤波器
- $b$ — 基线发放率
- $\mathbf{r}_{\text{hist}}$ — 历史脉冲向量

**关键操作：**
1. 使用 STA 的 SVD 分解初始化刺激滤波器（空间×时间分离）
2. 构建时间基函数（凸起/升余弦）参数化滤波器
3. 构建脉冲后历史滤波器基函数（含绝对不应期）
4. 使用 Jonathan Pillow 的 GLM 工具包进行最大似然拟合

**辅助函数：**
- `makeFittingStruct_GLM_Retina.m` — 初始化 GLM 参数结构

**依赖：** Jonathan Pillow 的 GLM 工具包

**输出：** 拟合的 GLM 参数（刺激滤波器 + 历史滤波器 + 基线率）

**复现要点：**
- GLM 相比 LN 模型增加了脉冲历史依赖
- 双线性（空间×时间分离）参数化大幅减少参数量
- 需要安装 Pillow 的 GLM 工具包

## 阶段六：模型预测与验证

### 步骤 6.1 — 生成预测
**对应脚本：** `script_108_RetinaData_Predictions.m`

**目标：** 使用各模型在测试集上生成预测发放率

**关键操作：**
1. 对每个模型（STA-LN, STC-LN, MNE, GLM），将测试刺激通过模型计算预测发放率
2. STA/STC 模型：投影 + 非线性映射
3. MNE 模型：使用拟合的 logistic 函数
4. GLM 模型：使用模拟方法生成条件强度函数

**输出：** 各模型的预测发放率时间序列

### 步骤 6.2 — 验证与比较
**对应脚本：** `script_109_RetinaData_Validation.m`

**目标：** 量化比较各模型的预测精度

**评价指标：**
- 相关系数（Pearson correlation）：预测发放率与实际 PSTH 之间
- 对数似然（log-likelihood）：模型对测试数据的拟合优度
- 信息量（bits/spike）：模型捕获的信息比率

**输出：** 各模型的性能指标对比

## 环境配置清单

| 项目 | 说明 |

| MATLAB 版本 | R2016a 或更高（推荐 R2020a+） |

## 复现检查点

完成每个阶段后，建议通过以下方式验证正确性：

- [ ] **阶段一**：检查刺激矩阵维度是否正确（T × N_X），响应向量为 0/1 二值
- [ ] **阶段二**：STA 应呈现中心-环绕拮抗的空间结构
- [ ] **阶段三**：STC 特征值谱中应有少数超出 Marchenko-Pastur 边界的显著值
- [ ] **阶段四**：MNE 的线性特征应与 STA 相似，二次特征应与 STC 方向一致
- [ ] **阶段五**：GLM 的刺激滤波器应与 STA 形态类似
- [ ] **阶段六**：GLM ≥ MNE ≥ STC ≥ STA（预测性能递增）
- [ ] **阶段七**：生成的图表应与论文中的 Figure 3-8 一致

## 注意事项

1. **默认分析细胞：** Cell 3，可通过修改 `icell` 变量分析其他细胞
2. **刺激配置：** 有三种空间-时间维度配置（`short2`, `short3`, `long`），默认使用 `short3`
3. **Jackknife 折数：** 默认 5 折，用于估计参数的标准误差
4. **运行顺序严格：** 必须按阶段顺序执行，后续脚本依赖前序脚本的输出
5. **GLM 工具包：** 需要单独安装，参见 [Pillow Lab GLM 工具包](https://github.com/pillowlab/GLMspiketools)
